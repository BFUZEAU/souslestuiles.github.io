# souslestuiles.github.io
Blog qui retrace l'avancer de la rénovation d'une maison familiale et partage nos "tips" pour réussir son projet.

Landing page ajoutée : `index.html` et ressources sous `assets/`.

Pour visualiser localement : ouvrez `index.html` dans votre navigateur ou servez le dossier avec un serveur statique depuis la racine du projet :

```bash
# depuis la racine du dépôt
python3 -m http.server 8000
# puis ouvrez http://localhost:8000
```

La landing est responsive, accessible et prête à être personnalisée (logo, images, textes, liens externes).
